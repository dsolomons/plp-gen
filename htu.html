Pardot Code Generator - What is a PCG?

What is the PCG?

Love scrolling through code and endlessly tweaking values to get things to display correctly? Me either! Hence the Pardot Code Generator came into existance. You tell it what to build, then sit back while the machine flashes lights and makes whirring sounds, and promptly spits out your code.


Generating your code

Step 1:
  • Download and install the colour picker browser plugin.
  • Chrome: http://www.colorzilla.com/chrome/
  • Firefox: http://www.colorzilla.com/firefox/
Step 2:
  • Select what code you want to create: Landing Page, Email, both.
Step 3:
  • Fill out the fields.
Step 4 - Colour Fields
  • Use the ColorZilla plugin to easily get the colour code from the prospects website.
  • For the colour fields, DO INCLUDE the # at the beginning of the code. EG
  • Correct = #ffffff
  • Incorrect = ffffff
Step 5:
    Press GENERATE to have the PCG create your code for you.

Using your code

After hitting Generate, a small group of magicians will incant ancient mantras and insert your values into the Landing Page and Email templates, saving you around 30.46-40.32 mins of work (no more, no less).

Previewing:
  • There is a PREVIEW button that will open a new tab and show you what your page looks like.
  • For Landing Pages, a fake form is included (your real form will be as created inside of Pardot).
Making a change to your code:
  • If you decide your preview doesn't look like what you wanted (wrong colours, typo etc), hit the "BACK" button in your browser. Select the tab that you were working on (LP, EDM, Both), and all of your fields will still be filled out for you. Make any changes desired, and hit the Generate button again.
Copying your code:
  • Inside the code box, at the top right hand side, there is "COPY" button. Pressing this will copy all of your code for you.
What to do with the code:
  • Log into your Pardot account. Create a new Landing Page or Email Template, and paste the html code into the appropriate screen.
  • For landing pages, when you create your landing page from this template, you will still need to select the correct form (the one shown in the preview is an example only).
